GitHub

Canvas+ (Canvas Plus) is a third party chrome extension that improves the user experience of students using Canvas. Developers cannot see any of your personal canvas data, as none of it is tracked in any way. Canvas+ does not interfere with the Canvas backend website- it won't ever cause a problem for your school administrators or technitians. We are in no way affilliated or endorsed by Instructure Inc.
