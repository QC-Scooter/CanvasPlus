By Students, for Students.

Canvas+ is an unofficial Google Chrome Extension that enhances your Canvas experience. We offer improved scrolling and more features to come that you're sure to enjoy.

Install for Chrome

Speed Boost

By loading Canvas links before you click them, you'll see loading speed improvements of 2x, 4x, or even 8x with this feature enabled!

Smart Scroll

Smart scroll saves you time when scrolling in course pages, discussions, and your modules/assignments list. The Scroll to Bottom and Back to Top buttons are very effective at navigating, especially in larger classes.

Search Bar

The Canvas search bar allows you to navigate between all of your courses from one page with ease.

Open Source

Anyone on the internet can examine the code of CanvasPlus with ease. Developers can even create Pull Requests on GitHub to modify the code. Collaboration is everything!

Canvas+ (Canvas Plus) is a third party chrome extension that improves the user experience of students using Canvas. Developers cannot see any of your personal canvas data, as none of it is tracked in any way. Canvas+ does not interfere with the Canvas backend website- it won't ever cause a problem for your school administrators or technitians. We are in no way affilliated or endorsed by Instructure Inc.
